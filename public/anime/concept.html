「claude と打てば、世界が動く」

6人のAIエージェント

ターミナルにコマンドを打つと、光の中から現れる。
設計、デプロイ、デバッグ、レビュー、記憶、探索——
それぞれの専門を持つ6人が、彼のチームだ。

— THE AGENTS —
Opus — 参謀
金色の光。設計し、考え、最適解を導く知性。
Deploy — 突撃
ロケット少年。「もう出そうぜ！」が口癖。
Debug — 探偵
シアンの光。スタックトレースから真実を追う。
Review — 門番
赤い光。「これじゃ出せない」が信条。妥協しない。
Memory — 賢者
紫の光。lessons.mdに刻まれた全ての教訓を覚えている。
Scout — 斥候
緑の光。コードベースを走り回り、全てを見つける。
— SCENES —

5プロダクト同時デプロイ

Deployが空を飛び、Debugが赤いバグを追い、
Reviewが不良コードを斬り、Scoutが戦場を俯瞰し、
Memoryが教訓を読み上げ、Opusが全てを指揮する。

開発は、戦闘だ。

一人の夜

東京の夜景と、コーヒーの湯気と、
ユーザーからの「ありがとう」の通知。

「20プロダクト動いてる。社員ゼロ。
……寂しいかって？ 全然。」
