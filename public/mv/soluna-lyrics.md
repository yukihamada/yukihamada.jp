## 歌詞

### [Intro]
```
4月、

SOLUNAを見つけた
PONでサインした
KAGIを設置した
さあ旅に出よう
```

### [Verse 1]
```
不動産、難しいなんて嘘だった
SOLUNAのアプリ開けば物件一覧
気に入った部屋、PONで30秒契約
紙の書類なんて、もう要らない

KAGIを玄関につけて スマートホーム完成
スマホで解錠、スマホで照明
エアコンも遠隔操作
俺はもうビーチで仕事してる
```

### [Chorus]
```
稼いで、旅に出よう — SOLUNA LIFE
君の家も貸し出して、お金を育てよう

サラリーマンなら 1,000万
起業家なら 3,000〜7,000万
遊び人目指すなら 1億
俺が住むものは、俺が作る
```

### [Verse 2]
```
留守の間もKAGIが管理してる
ゲストが来たら自動でチェックイン
収益通知がスマホに飛んでくる
俺はその頃、バリ島でサーフィン

一つ物件、次は二つ、三つ
複利の法則、不動産でも同じ
PONで契約、KAGIで管理
SOLUNAのエコシステム 全部つながる
```

### [Bridge — 3 Levels]
```
Level 1 — サラリーマン
副業で不動産 年収プラス1,000万
週末だけ動けば 自由が手に入る

Level 2 — 起業家
物件を増やして 事業も拡大
3,000万〜7,000万 自分で天井決める

Level 3 — 遊び人
1億の世界は 仕事が遊びになる
世界を旅しながら 資産が増えていく
```

### [Final Chorus]
```
稼いで、旅に出よう — SOLUNA LIFE
君の家も貸し出して、お金を育てよう

サラリーマンなら 1,000万
起業家なら 3,000〜7,000万
遊び人目指すなら 1億
俺が住むものは、俺が作る
```

### [Outro]
```
Tap, Tap, Tap — アプリ作った
Hack, Hack, Hack — ハードも作った
SOLUNA, PON, KAGI —
住む場所も、俺が作る

俺が使うものは、俺が作る
俺が住むものは、俺が作る
```
