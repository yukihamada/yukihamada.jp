## DAOとは何か？

DAO（Decentralized Autonomous Organization：分散型自律組織）は、ブロックチェーン技術を活用して、特定のリーダーや管理者を置かずに目的を達成する組織です。従来のトップダウン型の組織とは異なり、参加者全員が協力して運営を行い、貢献度に応じて報酬を得られる仕組みが特徴です。

2026年現在、DAOは単なる概念から実用的なツールへと進化しています。映画製作、地域創生、そしてソフトウェア開発など、さまざまな分野でDAOの活用が広がっています。特にソフトウェア開発においては、経営層の85%が今後3年以内にAIエージェントが業界標準になると予測する中、DAOによる分散型開発が新たな潮流として注目されています。

## EnablerDAOの使命

EnablerDAOは、グローバルなエンジニアコミュニティを構築し、オープンソースソフトウェアの開発を加速させることをミッションとしています。私たちは以下の価値観を大切にしています：

### 1. 透明性と公平性

すべてのコードはオープンソースとして公開され、誰でもレビュー・改善できます。貢献度はブロックチェーン上に記録され、改ざんできない形で保存されます。これにより、誰がどれだけ貢献したかが明確になり、公平な評価が実現します。

### 2. 分散型ガバナンス

EnablerDAOでは、重要な意思決定をEBRトークン保有者による投票で行います。プロジェクトの方向性、予算配分、新機能の優先順位などを、コミュニティ全体で決定します。中央集権的な管理者は存在せず、参加者全員が組織の未来を形作ります。

### 3. グローバルな協力

地理的な制約を超えて、世界中のエンジニアが協力できる環境を提供します。時差を活用した24時間開発体制、多言語対応、リモートワークを前提とした設計により、最高のタレントがどこからでも参加できます。

## EBRトークンの役割

EBR（Enabler）トークンは、EnablerDAOのエコシステムを支える中核的な要素です。単なる仮想通貨ではなく、貢献を可視化し、インセンティブを提供するためのツールとして機能します。

### トークンの獲得方法

- **コードコミット** — バグ修正、新機能の実装、パフォーマンス改善など、コードベースへの貢献に応じてEBRを獲得
- **コードレビュー** — 他の開発者のPRをレビューし、品質を向上させることで報酬を獲得
- **ドキュメント作成** — 技術文書、チュートリアル、APIリファレンスなどの執筆に対して報酬
- **バグ報告** — セキュリティ脆弱性や重大なバグの発見・報告に対してインセンティブ
- **コミュニティ貢献** — Discord/フォーラムでのサポート、メンタリング、イベント開催など

### トークンの使い道

- **ガバナンス投票** — DAOの重要な意思決定に参加する権利
- **プレミアム機能** — AI支援開発ツール、高度な分析ダッシュボードへのアクセス
- **報酬の獲得** — ステーキングによる追加報酬、プロジェクトマイルストーン達成時のボーナス
- **スキルアップ** — 有料コース、認定資格、エキスパートとの1on1セッションへのアクセス

## 具体的な貢献方法

EnablerDAOへの参加は簡単です。以下のステップで今日から始められます：

### ステップ1: CLIツールのインストール

```bash
curl -fsSL https://enablerdao.com/install.sh | bash
```

このコマンド一つで、EnablerDAO CLIツールがインストールされます。このツールを使って、プロジェクトの一覧表示、タスクの取得、貢献の提出などを行えます。

### ステップ2: プロジェクトを選ぶ

自分のスキルや興味に合ったプロジェクトを選びます。初心者向けの「Good First Issue」から、上級者向けの「Architecture」タスクまで、幅広い難易度のタスクが用意されています。

```bash
enablerdao projects list
enablerdao tasks --project=chatweb-ai --difficulty=beginner
```

### ステップ3: 作業を開始

タスクをクレームし、開発を開始します。GitHubとの統合により、通常の開発フローと同じように作業できます。

```bash
enablerdao claim-task 12345
git clone https://github.com/enablerdao/chatweb-ai
# コーディング、テスト、コミット
git push
enablerdao submit-pr 12345
```

### ステップ4: レビューと報酬

プルリクエストが承認されると、自動的にEBRトークンがウォレットに送信されます。貢献度に応じて、100 EBRから10,000 EBRまでの報酬が得られます。

## 現在進行中のプロジェクト

EnablerDAOでは、複数の実用的なプロジェクトが同時進行しています：

### chatweb.ai
マルチモデルAIチャットプラットフォーム。Claude、GPT-4、Gemini、ローカルLLMなどを統合し、ツール呼び出し（Web検索、天気、コード実行）にも対応。Rust + Lambda + DynamoDBで構築されています。

### DojoC
柔術（BJJ）コミュニティプラットフォーム。道場検索、試合情報、技術共有、トレーニングログなどを提供。Next.js 16にアップグレード済み。

### jiuflow-ssr
柔術選手データベース。100名以上のトップアスリートの情報を収録。Supabase + Axum + Askamaで構築され、SEO最適化されたSSRを実現。

### nanobot
Lambda上で動作するボット開発フレームワーク。LINE、Telegram、Webチャットに対応。音声認識（STT）、音声合成（TTS）、チャネル別プロンプト最適化を実装。

## DAOソフトウェア開発の未来

EnablerDAOが描く未来は、以下のような世界です：

**才能の民主化** — 学歴や所属組織に関係なく、誰でもスキルと貢献で評価される。アフリカの大学生でも、シリコンバレーのエンジニアと同じ土俵で戦える。

**継続的な報酬** — 一度書いたコードが長期的に使われ続ける限り、継続的に報酬を得られる。オープンソース開発が持続可能なキャリアになる。

**グローバルな協力** — 時差を活用し、24時間体制で開発が進む。問題が発見されたら、地球の反対側の誰かがすぐに修正する。

**AI + 人間の協働** — AIエージェントが単純作業を担当し、人間は創造的な問題解決に集中。EnablerDAO CLIがAIとの連携をサポート。

**透明性の徹底** — すべての意思決定、予算配分、コード変更がブロックチェーン上に記録され、永久に参照可能。
