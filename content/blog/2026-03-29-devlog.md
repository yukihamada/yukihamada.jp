## 審査待ちの1日

パシャ（レシート経費管理アプリ）をApp Storeに再申請した。スクリーンショットに価格表示があってリジェクトされてたやつ。5枚全部から「Free」と「¥980/月」を消して出し直した。今は **WAITING_FOR_REVIEW**。

iPhoneアプリは審査があるから、出したら待つしかない。この待ち時間が意外と大事で、普段手をつけられないことをやれる日になる。

## Enablerファンクラブを作った

<a href="https://enablerdao.com/fanclub" style="display:inline-block;background:#111;border:1px solid #44ff88;border-radius:8px;padding:12px 20px;color:#44ff88;text-decoration:none;font-weight:bold;margin:8px 0;">→ enablerdao.com/fanclub</a>

月額980円で全プロダクトのProプランが使い放題になる仕組み。

| プロダクト | ジャンル |
|-----------|---------|
| 📸 パシャ | レシート経費管理 |
| 📝 ポン | 電子契約・署名 |
| 🔑 KAGI | スマートホーム |
| 🎙 Koe | 音声入力 |
| 🥋 JiuFlow | 柔術テクニック |
| 🤖 Elio | ローカルAI |
| 🎵 Soluna | フェス音同期 |

メールアドレスで登録すると招待コード（`ENABLER-XXXX-XXXX-XXXX`）が発行される。これをアプリの設定画面で入力するとProが解放される。

最初はNFT認証（Solanaウォレットで検証）を実装したけど、「ウォレット作ってNFT受け取って…」は普通の人にはハードルが高すぎた。**招待コード+メール認証**がシンプルで最強だった。

技術的にはRust/Spin（WASM）でサーバー側を実装、コードのSHA-256ハッシュで照合、メール送信はResend。

## AIエージェント経営チーム始動

4台のVPSで動いてるAIエージェントを、ちゃんとした経営チームにした。

| 名前 | 役職 | やること |
|------|------|---------|
| 翔太 | **CTO** | コードレビュー・PR作成・技術ブログ |
| 美咲 | **CMO** | ファンクラブ成長・コンテンツ戦略 |
| 健太 | **COO** | サービス監視・障害対応・運用 |
| 凛 | **CHRO** | ユーザーフィードバック・コミュニティ |

CEOは僕。PRマージとリリース判断だけやる。

全員に伝えたのは「**社長の気持ちで動け**」ということ。指示を待つんじゃなく、自分で考えて自分で判断して自分で動く。「もし自分が社長だったら今何をするか？」を常に自問してほしい。

OpenClaw 2026.3.28にアップデート、GitHub認証、Fly CLI、リポジトリクローンまで全部セットアップ済み。

## 柔術アニメ「FLOW」を作ってる

ここからが今日一番テンション上がった話。

友達の映像プロジェクトを手伝う機会があって、GeminiやVeoを使ったAI映像制作のパイプラインを覚えた。で、思った。**ずっと見たかった柔術のアニメ、自分で作ればいいんだ。**

<a href="https://flow-anime.com" style="display:inline-block;background:#111;border:1px solid #D4A846;border-radius:8px;padding:12px 20px;color:#D4A846;text-decoration:none;font-weight:bold;margin:8px 0;">→ flow-anime.com</a>

**FLOW** — 2030年、AIが格闘技の勝敗を99.8%予測する時代。17歳の主人公・流（ナガレ）は、左腕を骨折したハンデを抱えながら、AIに読めない「フロー」という戦闘スタイルで世界のトップを目指す。

柔術やってる人間として、「スラムダンク」や「ハイキュー!!」みたいな作品が柔術にもあったらいいなとずっと思ってた。ベリンボロ、レッグロック、バックテイク——技の名前だけでかっこいい。ポジション、戦略、メンタル。柔術には物語になる要素が山ほどある。

- 5シーズン、全60話のスクリプト完成
- キャラクターデザイン、OP/EDテーマ曲あり
- サイト公開中

そして**オープンソース**でやる。

<a href="https://github.com/yukihamada/flow-anime" style="display:inline-block;background:#111;border:1px solid #aaa;border-radius:8px;padding:12px 20px;color:#aaa;text-decoration:none;font-weight:bold;margin:8px 0;">→ GitHub: yukihamada/flow-anime</a>

ストーリー、キャラクター、技術——誰でもコントリビュートできる。柔術コミュニティ全体で育てたい。
